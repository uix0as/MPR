# 사진 넣는 곳

이 폴더에 이미지를 넣으면 게임이 자동으로 불러옵니다.

## 파일 이름

아래 이름 중 하나로 저장하세요. PNG, JPG, JPEG를 지원합니다.

- `background.png` / `background.jpg`: 전체 배경
- `player.png` / `player.jpg`: 직접 움직이는 초록 O 말
- `o.png` / `o.jpg`: 일반 O 말
- `x.png` / `x.jpg`: X 말
- `wall.png` / `wall.jpg`: 벽
- `item.png` / `item.jpg`: ? 아이템

예를 들어 X 말 사진을 바꾸고 싶으면 이 폴더에 `x.png`를 넣고 게임을 다시 실행하면 됩니다.
`player.png`가 없으면 `o.png`를 플레이어 말에도 같이 쓰고 초록색 표시를 덧그립니다.

## 권장 크기

- 말/아이템/벽: 정사각형 이미지 권장, 예: 512x512
- 배경: 가로형 이미지 권장, 예: 1600x1000 이상

이미지가 없으면 게임은 기본 도형 디자인으로 표시됩니다.
